# Enumeration: CocoLabelYolo

Defined in: [constants/commonVision.ts:121](https://github.com/software-mansion/react-native-executorch/blob/d0d3e5b7a1d42b2e7bcd89806efcaf0b961974c9/packages/react-native-executorch/src/constants/commonVision.ts#L121)

COCO dataset class labels used by **YOLO** models for instance segmentation and object detection.

This enum is **0-indexed** (values start at 0) and contains exactly **80 classes** —
the standard COCO detection subset without gaps. This differs from [CocoLabel](CocoLabel.md),
which is 1-indexed with 91 classes and includes extra categories not present in the
YOLO label set.

Use this enum when working with YOLO models (e.g. `yolo26n-seg`).
For RF-DETR or SSDLite models, use [CocoLabel](CocoLabel.md).

## See

[CocoLabel](CocoLabel.md) for the RF-DETR / SSDLite variant
